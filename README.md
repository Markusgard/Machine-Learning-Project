# Machine Learning Project

## Project Description
In this project, I worked with two datasets: images of the Milky Way and Ebola outbreak data. I used K-means clustering to find patterns in the image data and tried different models like regression, neural networks, and LSTM to predict how the Ebola outbreak develops over time. The goal was to see how different methods work and how well they handle different types of data.
